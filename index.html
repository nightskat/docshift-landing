docshift

Format-preserving DOCX translation

Translate Word documents without losing bold, italic, tables, or headers. Bring your own API key.

npm install @docshift/core
Try the API →

How it works

01

Extract

Text separated from formatting into a FormatMap, ensuring no content is lost.

02

Translate

3-stage LLM pipeline (primer → translate → audit) for high-quality results.

03

Rebuild

Original run structure restored — formatting untouched and perfectly preserved.

Features

Format-safe

Bold, italic, headers, tables preserved with precision.

BYOK

Your API key, your cost, no lock-in. Full control over your expenses.

OSS

MIT license, zero telemetry. Privacy and transparency by design.

Universal

Any LLM via CoreProvider interface. Plug in any model you prefer.
