01

How it works

仕組み · shikumi
  1. Extract — text is separated from formatting into a FormatMap, so nothing is lost.
  2. Translate — a three-stage LLM pipeline (primer → translate → audit) makes the call.
  3. Rebuild — original run structure restored, formatting untouched.
02

Why it matters

意義 · igi

Most translation tools flatten Word documents. Bold disappears. Tables collapse. Headers turn into body text. That's because they translate the text and rebuild the shell. Akebi reverses it: the shell stays intact, only the text inside changes.

And because you bring your own LLM key, the cost is yours and the data path is yours. We never see either.

03

The metaphor

比喩 · hiyu
The akebi pod ripens, splits along its spine, and reveals the soft pulp inside — completely intact.

Akebi (木通) is a Japanese autumn fruit whose purple pod cracks open when ready. The pod splits. The fruit stays whole. That's the engineering metaphor.
